## Live App
https://neu-library-app-th7e.vercel.app

## GitHub Repository
https://github.com/avraignemartinez15/neu-library-app

## Features
- Google Sign-in restricted to @neu.edu.ph accounts
- Role-based access: regular user and admin
- Check-in with purpose, college, and employee flag
- Admin dashboard with day/week/month/custom date filters
- Filter by reason, college, employee type
- Visitor statistics displayed as stat cards
